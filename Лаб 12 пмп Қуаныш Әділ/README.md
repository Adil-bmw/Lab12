## Описание
Цель работы: изучить Fork/Join Framework и применить его для трёх задач:
1) Параллельное суммирование массива (RecursiveTask)
2) Параллельный поиск файлов по расширению (RecursiveAction)
3) Параллельная сортировка слиянием (RecursiveTask)

## Краткие результаты
- Параллельное суммирование: результат совпадает с последовательным.
- Поиск файлов: создаётся тестовая директория test_directory, задача находит файлы с расширением .txt.
- Параллельная сортировка: результаты совпадают с Arrays.sort(), выводится проверка корректности и времена выполнения.
